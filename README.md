# RankTracker v3.2
> Скрипт для автоматического отслеживания продаж и продлений рангов на сервере Arizona RP (SAMP / MonetLoader).  
> Автоматически считывает данные из диалога подтверждения и отправляет отчёт в Discord.

---

## 📋 Возможности

- Автоматическое считывание покупок и продлений рангов из диалога игры
- Отправка красивого embed-отчёта в Discord Webhook
- Умное форматирование сумм (500 K, 1 KK, 17 KK 500 K)
- Лог всех операций в файл
- Авто-масштабирование интерфейса под разрешение экрана (DPI)
- Автообновление скрипта с GitHub
- Поддержка **MoonLoader (PC)** и **MonetLoader (Mobile)**

---

## 🖥️ Интерфейс

Открывается командой `/fmenu`. Три вкладки:

| Вкладка | Содержимое |
|---|---|
| **Главная** | Последняя запись, кнопки отправки и теста |
| **Лог** | История операций из файла |
| **Настройки** | Имя менеджера, DPI, список команд |

---

## ⚙️ Установка

1. Скопируй файл `RankTracker.lua` в папку скриптов:
   - **MoonLoader (PC):** `moonloader/`
   - **MonetLoader (Mobile):** `/Android/media/com.arizona.game.git/monetloader/RangTreckerDS.lua`

> Если лаунчер х32 то папка будет `com.arizona.game`
> Если лаунчер х64
`com.arizona.game.git`

2. Открой файл и вставь свой Discord Webhook:
```lua
local DISCORD_WEBHOOK = "https://discord.com/api/webhooks/..."
```

3. Укажи своё имя менеджера (или смени командой `/rname`):
```lua
local MANAGER_NAME = "Твой_Ник"
```

4. Запусти скрипт — папка `RankTracker/` создастся автоматически.

## 🔄 Автообновление

Скрипт проверяет файл `Update.json` на GitHub и при наличии новой версии скачивает и перезапускает себя автоматически.

Формат `Update.json`:
```json
{
    "version": "3.2",
    "url": "https://raw.githubusercontent.com/USER/REPO/main/RankTracker.lua"
}
```

> Замени `USER` и `REPO` в коде скрипта на свои данные GitHub.

## 📌 Поддерживаемые форматы сумм

Скрипт автоматически понимает все форматы цен из диалога игры:

| Формат в игре | Результат |
|---|---|
| `КК 1` | 1 KK |
| `КК 35` | 35 KK |
| `КК 3 * 500.000` | 3 KK 500 K |
| `КК 17 * 500.000` | 17 KK 500 K |
| `100.100` | 100 K |
